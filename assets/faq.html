FAQ

Respostas para as perguntas mais comuns sobre nosso projeto

Como minha ONG pode utilizar a plataforma?

Basta entrar em contato conosco para criar uma conta. Oferecemos suporte completo para implementação.

Quais os critérios para um paciente ser atendido?

Desenvolvemos critérios baseados em: urgência do caso, condições socioeconômicas e idade.

A plataforma é gratuita para ONGs?

Sim, a plataforma é completamente gratuita para organizações sem fins lucrativos.

Como os voluntários são cadastrados?

Profissionais podem se cadastrar diretamente na plataforma, informando especialidade e disponibilidade.
