# SorrisON - Plataforma de Gestão de Triagem e Voluntários

## 📋 Sobre o Projeto
O SorrisON é uma plataforma inteligente de gestão para triagem odontológica e voluntários, otimizando o atendimento a comunidades carentes através da tecnologia.

## 🎯 Objetivo
Conectar sorrisos e transformar vidas através de uma solução completa para organizar atendimentos odontológicos voluntários de forma eficiente e escalável.

## 🚀 Funcionalidades Principais
- **Cadastro de Pacientes**: Sistema estruturado para registro completo
- **Triagem Inteligente**: Critérios automáticos de priorização
- **Rede de Voluntários**: Cadastro de dentistas com especialidades
- **Relatórios e Métricas**: Indicadores de impacto social
